# Tech Stack: Spotify 2024 Discovery | Elite

## Backend Layer
- **Framework**: FastAPI (Python 3.10+)
- **Validation**: Pydantic v2
- **Logging**: Structured `logging` with custom config
- **ML Inference**: Scikit-Learn (Joblib for persistence)
- **Data Manipulation**: Pandas, NumPy
- **Server**: Uvicorn

## Frontend Layer
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript (Strict Mode)
- **Styling**: Tailwind CSS v3
- **Animations**: Framer Motion
- **Visualizations**: Recharts
- **Icons**: Lucide React

## Development Tools
- **Version Control**: Git
- **Package Managers**: npm (Frontend), pip (Backend)
- **Standards**: Google-style docstrings, ESLint (Next.js defaults)
